Oto **tabela z ciekawymi komendami**, które możesz uruchomić przez **Win + R** w Windows 10. To są wbudowane narzędzia, o których wiele osób nie wie:

| **Polecenie**      | **Opis działania**                                                                          |
| ------------------ | ------------------------------------------------------------------------------------------- |
| `psr.exe`          | **Problem Steps Recorder** – nagrywa kroki użytkownika wraz ze zrzutami ekranu.             |
| `winver`           | Pokazuje wersję systemu Windows.                                                            |
| `dxdiag`           | **DirectX Diagnostic Tool** – szczegółowe informacje o sprzęcie i sterownikach graficznych. |
| `cmd`              | Uruchamia wiersz polecenia (Command Prompt).                                                |
| `powershell`       | Uruchamia Windows PowerShell.                                                               |
| `regedit`          | Otwiera edytor rejestru.                                                                    |
| `msconfig`         | **System Configuration** – ustawienia startu systemu, boot, usługi.                         |
| `taskmgr`          | Otwiera **Menedżera zadań**.                                                                |
| `control`          | Uruchamia klasyczny Panel sterowania.                                                       |
| `services.msc`     | Otwiera listę usług systemowych, które możesz włączać/wyłączać.                             |
| `compmgmt.msc`     | **Computer Management** – narzędzia administracyjne w jednym miejscu.                       |
| `eventvwr`         | **Event Viewer** – podgląd dzienników zdarzeń systemowych.                                  |
| `perfmon`          | **Performance Monitor** – monitorowanie wydajności systemu.                                 |
| `mrt`              | Uruchamia narzędzie Microsoft do usuwania złośliwego oprogramowania.                        |
| `osk`              | **On-Screen Keyboard** – klawiatura ekranowa.                                               |
| `magnify`          | **Lupa** – powiększenie ekranu.                                                             |
| `narrator`         | Uruchamia Narratora (czytnik ekranu).                                                       |
| `snippingtool`     | Uruchamia starsze narzędzie do wycinania (Snipping Tool).                                   |
| `cleanmgr`         | **Disk Cleanup** – oczyszczanie dysku ze zbędnych plików.                                   |
| `optionalfeatures` | Otwiera okno **funkcji systemu Windows**, które można włączać/wyłączać.                     |
| `appwiz.cpl`       | Lista zainstalowanych programów (Panel sterowania → Programy i funkcje).                    |
| `ncpa.cpl`         | Otwiera ustawienia kart sieciowych.                                                         |
| `sysdm.cpl`        | Właściwości systemu (m.in. zmienne środowiskowe, wydajność).                                |
| `control printers` | Otwiera panel drukarek i urządzeń.                                                          |
| `inetcpl.cpl`      | Otwiera ustawienia internetowe (Internet Options).                                          |
| `main.cpl`         | Otwiera ustawienia myszy.                                                                   |
| `hdwwiz.cpl`       | Menedżer urządzeń (skrót alternatywny).                                                     |
| `diskmgmt.msc`     | **Zarządzanie dyskami** – partycje, formatowanie itp.                                       |
| `powercfg.cpl`     | Ustawienia planów zasilania.                                                                |
| `timedate.cpl`     | Ustawienia daty i godziny.                                                                  |
| `shell:startup`    | Otwiera folder autostartu użytkownika.                                                      |
| `shell:sendto`     | Otwiera folder "Wyślij do".                                                                 |
